## Messages clés

- Claude Code pilote l'ordinateur entier : écrit des fichiers, les lance, guide l'utilisateur — sans taper une ligne de code
- Réaction à la première utilisation : "je viens de découvrir la bombe atomique"
- Un entrepreneur sans compétences en code peut créer un SaaS complet avec l'IA en 2026
- Kevin ne tape plus une seule ligne de code depuis des mois
- ChatGPT est jugé inférieur à Claude pour le code (consensus des deux invités)
- L'IA va supprimer 50 % des emplois dans les 5 prochaines années (selon eux)
- Les formations IA en ligne sont jugées à 90 % du bullshit
- Les enfants de Johann (8 et 9 ans) utilisent déjà Claude Code

## Mises en garde importantes

- Ne pas tomber dans le piège : accumuler des outils sans rien produire
- L'erreur commune = toujours vouloir améliorer l'outil plutôt que de livrer
- Se poser la question chaque semaine : "j'ai produit quoi comme valeur ?"
- Le plus gros danger de l'IA : ne pas perdre son job, mais perdre sa vie (consensus)
- Méfiance sur la souveraineté des données : éviter de tout confier à une méga-société
